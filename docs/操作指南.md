## 二、滚轮功能分配规则

**核心原则：滚轮在任何时刻只有一种职责，不会冲突。**

### 默认状态（浏览模式）

| 滚轮位置 | 行为 | 说明 |
|---------|------|------|
| 在地图区域 | **页面滚动** | 驱动滚动叙事，step 卡片依次激活 |
| 在文字区域 | **页面滚动** | 同上 |
| 在空白区域 | **页面滚动** | 同上 |

### 地图激活状态（缩放模式）

| 章节 | 激活方式 | 滚轮行为 |
|------|---------|---------|
| **章节1** — 全球脉冲地图 | 点击地图上的**海洋/空白区域** | 滚轮→缩放地图；页面不动 |
| **章节5** — 疫苗脉冲地图 | 点击地图上的**海洋/空白区域** | 滚轮→缩放地图；页面不动 |

### 退出缩放模式

| 操作 | 效果 |
|------|------|
| 点击右侧文字区域 | 退出缩放，滚轮恢复页面滚动 |
| 滚动到其他章节 | 自动退出缩放模式 |

## 三、鼠标操作详解

### 章节1 — 全球传播脉冲地图

| 操作 | 目标 | 行为 |
|------|------|------|
| 点击国家 | 地图上的国家 | 选中/取消该国（联动其他图表），**不缩放** |
| 点击海洋 | 地图上的空白海域 | **激活缩放模式**（红色边框+提示条出现） |
| 滚轮（默认） | 地图区域 | 页面滚动叙事 |
| 滚轮（缩放模式） | 地图区域 | 地图缩放（1x~8x） |
| 拖拽（缩放模式） | 地图区域 | 平移地图 |
| 悬浮 | 国家上方 | 显示 Tooltip：确诊/死亡/政策强度 |
| 搜索框 | 左上角 | 输入中文名或ISO代码定位国家 |
| 点击右侧文字 | 文字区域 | **退出缩放模式** |

### 章节2 — 政策时间线

| 操作 | 目标 | 行为 |
|------|------|------|
| 滚轮 | 任意位置 | 页面滚动叙事（时间线不支持缩放） |
| 悬浮 | 曲线上方 | 十字线+双数据点悬浮，显示当日详细数据 |
| 国家切换 | 自动 | step 切换时自动切换对比国家 |

### 章节3 — 最佳干预窗口

| 操作 | 目标 | 行为 |
|------|------|------|
| 滚轮 | 任意位置 | 页面滚动叙事 |
| 拖动滑块 | 底部滑块 | 调整政策偏移天数（-30 ~ +30 天） |
| 自动演示 | step3 激活 | 自动从 -30 匀速滑到 0 天 |

### 章节4 — 波次雷达图

| 操作 | 目标 | 行为 |
|------|------|------|
| 滚轮 | 任意位置 | 页面滚动叙事 |
| 点击波次按钮 | 图表顶部 | 切换显示第 1/2/3/4/5 波 |
| 点击图例 | 右侧面板 | 添加/移除对比国家 |
| 悬浮 | 多边形/数据点 | 显示该维度具体数值 |

### 章节5 — 疫苗脉冲地图

| 操作 | 目标 | 行为 |
|------|------|------|
| 点击国家 | 地图上的国家 | 选中/取消该国，**不缩放** |
| 点击海洋 | 地图空白海域 | **激活缩放模式** |
| 滚轮（缩放模式） | 地图区域 | 地图缩放（1x~6x） |
| 拖拽（缩放模式） | 地图区域 | 平移地图 |
| 点击右侧文字 | 文字区域 | **退出缩放模式** |

### 章节6 — 政策热力矩阵

| 操作 | 目标 | 行为 |
|------|------|------|
| 滚轮 | 任意位置 | 页面滚动叙事 |
| 点击单元格 | 矩阵格子 | 高亮采用该组合的国家（联动脉冲地图） |
| 悬浮 | 单元格 | 显示组合名称、国家数、平均评分 |
| 切换指标 | 顶部按钮 | 确诊下降率 / 死亡率下降率 / 综合评分 |

## 五、图表联动规则

所有图表通过 Pinia Store 共享状态，点击任一图表会影响其他图表：

```
脉冲地图 点击国家 → 选中 → 时间线切换国家 + 雷达图更新 + 疫苗地图高亮
雷达图   添加国家 → 选中 → 脉冲地图高亮 + 时间线切换
热力矩阵 点击格子 → 高亮国家组 → 脉冲地图橙色高亮 + 疫苗地图高亮
疫苗地图 点击国家 → 选中 → 时间线切换 + 雷达图更新
```

**注意**：手动选中任何国家后，时间线不再自动轮换国家。取消所有选中后恢复自动轮换。
