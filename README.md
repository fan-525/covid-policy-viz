# 🦠 病毒与政策：一场全球博弈

> 一个关于 COVID-19 全球大流行的交互式数据可视化文档，探索政策响应时机如何深刻影响疫情走向。

## 📖 项目简介

本项目通过 **滚动叙事（Scrollytelling）** 的方式，将复杂的疫情数据与政策响应分析以直观、交互式的图表呈现给读者。用户在滚动页面的过程中，地图、时间线、反事实推演等可视化图表会随之动态变化，帮助读者深入理解"政策时机"这一核心命题。

### 核心问题

> 同样面对一种全新的病毒，为什么有些国家的应对如此高效，而另一些却步履维艰？答案很大程度上与一个因素有关——**政策的响应时机**。

## 📊 可视化图表

### 图表 1：全球传播脉冲地图（GlobalPulseMap）

- **技术**: D3.js + TopoJSON + Canvas 粒子系统
- **功能**:
  - Natural Earth 投影世界地图底图，支持 D3 zoom 缩放与平移
  - 各国地理中心渲染脉冲光圈动画，半径编码当日新增病例数
  - Canvas 层渲染跨国传播流动粒子，颜色随变异株时期变化（原始株 → Alpha → Delta → Omicron）
  - **悬浮时显示中文 Tooltip**：国家名称、当日新增确诊、累计确诊、累计死亡、5 项政策强度等级
  - 支持国家搜索功能（可按中英文名称搜索）
  - **地图交互模式**：点击海洋空白区域激活缩放——滚轮缩放地图+拖拽平移（此时页面不滚动）；点击国家仅选中/取消；点击右侧文字退出缩放恢复滚动叙事
  - 点击国家可选中/取消，联动时间线图表
  - 响应滚动时间变化，动态更新脉冲数据

### 图表 2：政策时机竞赛时间线（PolicyTimeline）

- **技术**: D3.js 双 Y 轴叠加图 + 阶梯面积图
- **功能**:
  - **上方区域**：日新增病例面积图（红色，左侧 Y 轴） + **综合政策指数阶梯线**（蓝色，右侧 Y 轴 0-100%），双轴直接叠加对比
  - **下方区域**：5 维度政策强度阶梯面积图（封锁/检测/口罩/疫苗/边境），使用 `curveStepAfter` 阶梯函数精确表达离散政策变化
  - 十字线 + 双数据点悬浮交互（红色 = 病例，蓝色 = 政策指数）
  - 自动标注"最佳干预窗口"（橙色竖条）
  - 政策变化到拐点的连接虚线，直观展示时间差
  - 关键政策变化节点标注（如"封锁↑""口罩↑"）
  - 反事实推演模拟曲线（紫色），支持 -30 到 +30 天偏移滑块
  - 国名中文标签 + 分项图例

### 图表 3：疫情波次雷达图（WaveRadarChart）

- **技术**: D3.js 雷达图 + 多边形面积排序
- **功能**:
  - 5 个维度评估疫情波次：峰值、规模、爆发强度、持续时间、致命性
  - 支持切换 1-5 波次，动态更新各国表现
  - 多国对比（最多同时显示 5-6 个国家）
  - 悬浮显示详细数据（具体数值）
  - 按多边形面积从小到大排序，避免大面积图形遮挡
  - 支持添加/移除国家，交互式对比分析  


### 图表 4：全球疫苗接种脉冲地图（VaccinePulseMap）

- **技术**: D3.js + TopoJSON 地图 + 脉冲动画
- **功能**:
  - 基于人口计算接种率 = 累计接种剂次 / 人口 × 100%
  - 脉冲半径映射：接种率 0% → 3px，100% → 12px
  - 悬浮显示国家名称 
  - 点击国家可选中，联动其他图表
  - 支持地图缩放和平移

### 图表 5：政策组合热力矩阵（PolicyHeatmap）

- **技术**: D3.js 热力图 + 分位数颜色映射
- **功能**:
  - 5×5 政策维度组合矩阵（封锁、检测、口罩、疫苗、边境）
  - 颜色深浅表示采取该组合的国家在疫情控制上的平均表现（绿色越深效果越好）
  - 三种评价指标切换：确诊下降率、死亡率下降率、综合评分
  - 点击单元格高亮采取该组合的国家，联动地图
  - 悬浮显示：组合名称、采用国家数、平均效果评分、推荐高亮国家（前5个）

## 🗄 数据说明

### merged_covid_data.json

按 ISO 3166-1 三字母代码索引，每个国家包含时序数据数组：

| 字段 | 说明 |
|------|------|
| `location` | 国家英文名称 |
| `date` | 日期（YYYY-MM-DD） |
| `new_cases` | 当日新增确诊 |
| `new_cases_smoothed` | 7 日平均新增确诊 |
| `total_cases` | 累计确诊 |
| `new_deaths` | 当日新增死亡 |
| `total_deaths` | 累计死亡 |
| `policy_lockdown` | 封锁政策强度（0-3） |
| `policy_testing` | 检测政策强度（0-3） |
| `policy_mask` | 口罩政策强度（0-3） |
| `policy_vaccine` | 疫苗政策强度（0-3） |
| `policy_border` | 边境管控强度（0-3） |

- **覆盖范围**: 181 个国家，2020-01-05 至 2023-02-28
- **数据来源**: Our World in Data (OWID)，经 `scripts/preprocess.py` 清洗标准化

### wave_features.json

各国疫情波次特征数据，包含波次起止日期、峰值、持续天数、病死率等。

## 🔗 全局状态联动

项目通过 Pinia Store (`src/stores/global.js`) 实现图表间的联动：

- `selectedCountries` — 选中国家列表，地图点击 ↔ 时间线切换
- `currentTime` — 当前时间点，滚动驱动 → 图表动画
- `timeRange` — 时间范围，章节切换 → 图表更新
- `counterfactualShift` — 反事实推演偏移天数
- `activeChart` — 当前图表类型（`pulseMap` | `timeline` | `timelineFocus`）
- `currentWave` — 当前波次
- `currentPolicyMetric` — 热力矩阵当前指标
